# AGENTS.md — machin-rag

Agent-first local RAG (machin/MFL + Qdrant + hash embedder). Humans state intent; agents drive the CLI/API.

## Philosophy

- **JSON always** on CLI stdout; errors on stderr with semantic exit codes.
- **No BYOK** — local hash embedder + local Qdrant.
- **Single binary** — Vue UI embedded at compile time (`ui/index.html` → `src/ui_gen.src`).
- **Do not** grow SSO, cloud vector vendors, paid embedding APIs, or admin panels — keep the agent surface small.
- **Formats** — md / txt / csv / json / ndjson / **docx** / **pdf**. OCR opt-in for images-only (see `docs/ocr-policy.md`).

| `src/docx.src` | OOXML walk (images/tables/links/headers/notes) |
| `src/pdf.src` | PDF text-layer (+ OCR empty pages) |
| `src/ocr.src` | OCR policy + Tesseract dlopen soft dep |
| `native/mr_zip.c` | ZIP reader + sha256 (zlib + libcrypto) |
| `native/mr_pdf.cpp` | poppler-cpp C ABI wrapper |
| `native/mr_ocr.c` | Tesseract via `dlopen` (optional) |
| `vendor/poppler/cpp/` | poppler-cpp headers (runtime `.so` system) |
| `docs/ocr-policy.md` | text-only / mixed / images-only decision |
| `scripts/smoke.sh` | CI smoke md/csv/json/docx/pdf |
| `scripts/eval_hash.py` | hash embedder lexical recall@k |
| `scripts/eval_synonyms.py` | synonym-stress suite (ONNX gate) |
| `docs/embedder-roadmap.md` | why ONNX is paused |
| `src/embed.src` | `embed_json`, `point_id` (hash-sha256-v1, dim 64) |
| `src/qdrant.src` | REST client + batch `upsert_chunks` |
| `src/api.src` | HTTP handlers + `out_ok` / `out_err` |

## Config precedence

1. `~/.machin-rag/config.json` (or `$MACHIN_RAG_CONFIG`)
2. Env: `QDRANT_URL`, `MACHIN_RAG_*`
3. CLI: `-Q` / `--qdrant-url`, `-F` / `--format`

## Commands agents use

```bash
./machin-rag help-json
./machin-rag status
./machin-rag ingest -c docs -t "…"
./machin-rag ingest -c docs -f path.md
./machin-rag ingest -c docs -f rows.csv
./machin-rag ingest -c docs -f items.json -F json
./machin-rag ingest -c docs -f notes.docx
./machin-rag ingest -c docs -f paper.pdf
./machin-rag query -c docs -q "…" -k 5
./machin-rag collections
./machin-rag serve -port 7091
```

## Extend

- **New format** — parser in `parsers.src` (or dedicated `*.src`), register in `chunk.src` detect + `ingest_*`.
- **DOCX** — walk: headings/tables/images/hyperlinks; headers/footers/footnotes/endnotes. Config: `.docx.image_stubs`. Needs zlib + OpenSSL crypto.
- **PDF** — text-layer via poppler-cpp; OCR empty pages when `ocr.enabled`. Needs `libpoppler-cpp0v5`.
- **OCR** — soft dep (`dlopen` libtesseract). Images-only docs refuse unless enabled. See `docs/ocr-policy.md`.
- **Normalize** — set `normalize.template` in config (e.g. `"{{title}}: {{body}}"`); empty fields dropped when `drop_empty`.
- **New API route** — handler in `api.src`, `route(...)` in `make_router` (`main.src`).
- **Better embedder** — keep `embed_json(text) -> json-array` signature; swap body.
- **Rebuild** — `./build.sh` after any `src/` or `ui/` change.

## CI

GitHub Actions (`.github/workflows/smoke.yml`) builds the binary and runs `scripts/smoke.sh` on every push — so PDF/DOCX/md regressions fail in PR before merge. Pages deploy is separate (GitHub Pages).

## Gotchas

- Qdrant must be up (`docker compose up -d`) or `status`/`ingest`/`query` return exit `106`.
- Router matches exact paths; query strings are stripped in `dispatch_rag`.
- Hash embedder is **not** semantic ML — fine for agent notes/docs MVP, not for legal discovery.
- Machin rejects non-empty struct slice literals — build `[]Chunk` via `append`.
